# 网吧USB管理软件 - 预编译部署指南

## 📋 概述
本文档提供网吧USB管理软件的预编译部署方案，适合在网吧环境直接运行已编译的程序，无需在网吧机器上安装开发工具。

## 🛠️ 第一步：在开发机上编译程序

### 1.1 环境要求（开发机）
- Windows 10/11
- Visual Studio 2019+ 或 .NET Framework 4.7.2 SDK
- 管理员权限

### 1.2 编译方法

#### 方法一：使用Visual Studio（推荐）
```batch
# 打开解决方案
双击 CyberCafeUsbManager.sln

# 编译步骤：
1. 选择"Release"配置
2. 选择"Any CPU"平台
3. 点击"生成" → "重新生成解决方案"
```

#### 方法二：使用命令行编译
```powershell
# 使用MSBuild（Visual Studio自带的）
cd "C:\Users\Administrator\Desktop\cybercafe-usb-manager"
"C:\Program Files (x86)\Microsoft Visual Studio\2019\BuildTools\MSBuild\Current\Bin\MSBuild.exe" CyberCafeUsbManager.sln /p:Configuration=Release /p:Platform="Any CPU"

# 或使用dotnet（需要.NET SDK）
dotnet build CyberCafeUsbManager.sln -c Release
```

### 1.3 验证编译结果
编译成功后，检查以下目录：
```
CyberCafeUsbManager\bin\Release\
├── CyberCafeUsbManager.exe          # 主程序
├── CyberCafeUsbManager.exe.config   # 配置文件
├── Newtonsoft.Json.dll              # JSON库
├── log4net.dll                      # 日志库
├── TaskScheduler.dll                # 任务调度库
└── *.pdb                            # 调试符号文件（可选）
```

### 2.2 创建部署脚本

#### 安装服务.bat
```batch
@echo off
chcp 65001 >nul
echo ========================================
echo 网吧USB管理软件 - 服务安装工具
echo ========================================
echo.

REM 检查管理员权限
net session >nul 2>&1
if %errorLevel% neq 0 (
    echo ❌ 请以管理员身份运行此脚本
    echo 右键点击 -> "以管理员身份运行"
    pause
    exit /b 1
)

echo ✅ 管理员权限确认
echo.

REM 切换到脚本所在目录
cd /d "%~dp0"

REM 检查主程序是否存在
if not exist "..\主程序\CyberCafeUsbManager.exe" (
    echo ❌ 找不到主程序: ..\主程序\CyberCafeUsbManager.exe
    pause
    exit /b 1
)

echo 正在安装服务...
echo.

REM 方法一：使用installutil（如果可用）
where installutil >nul 2>&1
if %errorLevel% equ 0 (
    echo 使用installutil安装服务...
    installutil "..\主程序\CyberCafeUsbManager.exe"
    
    if %errorLevel% equ 0 (
        echo ✅ 服务安装成功
    ) else (
        echo ⚠️  installutil安装失败，尝试方法二...
        goto :method2
    )
) else (
    echo ⚠️  未找到installutil，使用方法二...
    goto :method2
)

goto :start_service

:method2
echo.
echo 使用方法二：SC命令创建服务...
sc create CyberCafeUsbManager binPath= "%~dp0..\主程序\CyberCafeUsbManager.exe" start= auto DisplayName= "CyberCafe USB Manager"

if %errorLevel% equ 0 (
    echo ✅ 服务创建成功
) else (
    echo ❌ 服务创建失败
    pause
    exit /b 1
)

:start_service
echo.
echo 启动服务...
sc start CyberCafeUsbManager
timeout /t 3 /nobreak >nul

echo.
echo 检查服务状态...
sc query CyberCafeUsbManager

echo.
echo ========================================
echo 安装完成！
echo ========================================
echo.
echo 下一步操作：
echo 1. 检查服务是否正常运行
echo 2. 查看事件查看器日志
echo 3. 插入USB设备测试识别功能
echo.
echo 按任意键退出...
pause >nul
exit /b 0
```

#### 卸载服务.bat
```batch
@echo off
chcp 65001 >nul
echo ========================================
echo 网吧USB管理软件 - 服务卸载工具
echo ========================================
echo.

REM 检查管理员权限
net session >nul 2>&1
if %errorLevel% neq 0 (
    echo ❌ 请以管理员身份运行此脚本
    pause
    exit /b 1
)

echo ✅ 管理员权限确认
echo.

REM 停止服务
echo 停止服务...
sc stop CyberCafeUsbManager 2>nul
timeout /t 2 /nobreak >nul

REM 卸载服务
echo 卸载服务...
where installutil >nul 2>&1
if %errorLevel% equ 0 (
    installutil /u "%~dp0..\主程序\CyberCafeUsbManager.exe" 2>nul
) else (
    sc delete CyberCafeUsbManager 2>nul
)

echo.
echo 清理完成！
echo.
echo 按任意键退出...
pause >nul
exit /b 0
```

## 🚀 第三步：网吧部署步骤

### 3.1 部署前准备
1. **网吧机器要求**：
   - Windows 10 系统
   - .NET Framework 4.7.2 或更高版本
   - 管理员权限（用于安装服务）

2. **检查.NET Framework**：
   ```batch
   reg query "HKLM\SOFTWARE\Microsoft\NET Framework Setup\NDP\v4\Full" /v Release
   ```
   Release值 >= 461808 表示已安装.NET 4.7.2

### 3.2 部署流程

```batch
# 1. 复制部署包到网吧机器
# 例如：C:\网吧软件\USB管理软件\

# 2. 运行安装服务（管理员权限）
右键点击"安装服务.bat" → "以管理员身份运行"

# 3. 验证安装
# 查看服务是否正常运行
sc query CyberCafeUsbManager

# 4. 测试USB设备识别
# 插入Logitech、Razer等USB设备
# 查看事件查看器日志
eventvwr.msc
```

### 3.3 验证部署成功
1. **服务状态**：服务应显示"正在运行"
2. **事件日志**：在Windows事件查看器中查看"应用程序"日志
3. **功能测试**：插入USB设备，检查是否自动识别并安装驱动

## 🔧 第四步：网吧环境特殊配置

### 4.1 顺网系统集成
如果网吧使用顺网管理系统，可能需要：

1. **开机启动配置**：
   ```batch
   # 创建开机任务
   schtasks /create /tn "CyberCafeUsbManager" /tr "C:\网吧软件\USB管理软件\主程序\CyberCafeUsbManager.exe" /sc onstart /ru SYSTEM
   ```

2. **注册表配置**（如果需要自启动）：
   ```batch
   reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" /v "CyberCafeUsbManager" /t REG_SZ /d "C:\网吧软件\USB管理软件\主程序\CyberCafeUsbManager.exe" /f
   ```

### 4.2 驱动安装配置
如果网吧需要预装特定驱动：

1. **创建驱动包**：
   ```
   驱动程序/
   ├── Logitech/
   │   ├── GamingSoftware.exe
   │   └── Drivers/
   ├── Razer/
   │   ├── SynapseInstaller.exe
   │   └── Drivers/
   └── SteelSeries/
       └── EngineInstaller.exe
   ```

2. **驱动安装脚本**：
   ```batch
   @echo off
   echo 安装Logitech驱动...
   start /wait ".\驱动程序\Logitech\GamingSoftware.exe" /S
   
   echo 安装Razer驱动...
   start /wait ".\驱动程序\Razer\SynapseInstaller.exe" /S
   
   echo 驱动安装完成！
   pause
   ```

## 📊 第五步：监控与维护

### 5.1 日志查看
程序日志位于：
- **Windows事件查看器**：应用程序和服务日志
- **程序日志文件**：`C:\ProgramData\CyberCafeUsbManager\logs\`（如果配置了文件日志）

### 5.2 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 服务无法启动 | .NET Framework缺失 | 安装.NET 4.7.2+ |
| USB设备不识别 | 权限不足 | 以管理员运行 |
| 驱动安装失败 | 驱动文件缺失 | 检查驱动包 |
| 程序崩溃 | 依赖库缺失 | 检查所有DLL文件 |

### 5.3 更新部署
1. 停止服务：`sc stop CyberCafeUsbManager`
2. 备份原有文件
3. 复制新版本文件
4. 启动服务：`sc start CyberCafeUsbManager`
