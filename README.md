# 网吧USB设备管理软件

## 项目概述
为网吧环境开发的USB键盘鼠标自动识别与管理软件，实现设备即插即用体验。

## 核心功能
1. **自动识别** - 识别插入的USB键盘鼠标品牌型号
2. **驱动安装** - 自动安装对应品牌驱动程序
3. **程序启动** - 自动启动品牌配套软件（如雷蛇Synapse、罗技G Hub等）

## 目标系统
- Windows 10（用户已确认）

## 模块设计
### 1. 设备识别模块
- 使用WMI查询USB设备信息
- 解析VID/PID匹配品牌数据库
- 本地数据库 + 在线更新机制

### 2. 驱动管理模块
- 本地驱动库管理
- 在线驱动下载（可选）
- 驱动签名验证（Windows 10要求）

### 3. 程序管理模块
- 品牌程序路径映射
- 自动检测已安装软件
- 静默启动控制

### 4. 配置管理模块
- 白名单/黑名单管理
- 日志记录系统
- 设置界面

## 关键技术挑战
1. **权限要求** - 需要管理员权限安装驱动
2. **驱动签名** - Windows 10强制要求数字签名
3. **系统还原** - 网吧环境可能有还原系统
4. **兼容性** - 不同品牌驱动安装方式各异
5. **维护成本** - 新设备需要更新数据库
