网吧USB管理软件 - 部署包
============================

## 概述
本部署包包含网吧USB管理软件的所有必要文件，用于在网吧环境中自动识别USB键盘鼠标并安装驱动。

## 文件结构
```
网吧USB管理软件_部署包/
├── 主程序/                    # [需手动添加] 编译好的程序文件
│   ├── CyberCafeUsbManager.exe          # 主程序
│   ├── CyberCafeUsbManager.exe.config   # 配置文件
│   ├── Newtonsoft.Json.dll              # JSON库
│   ├── log4net.dll                      # 日志库
│   └── TaskScheduler.dll                # 任务调度库
├── 配置文件/                  # 预设配置文件
│   ├── usb_device_database.json         # USB设备数据库
│   └── app.config.example               # 配置文件示例
├── 脚本工具/                  # 安装和管理脚本
│   ├── 安装服务.bat          # 安装Windows服务
│   ├── 卸载服务.bat          # 卸载Windows服务
│   ├── 控制台测试.bat        # 控制台调试模式
│   ├── 检查状态.bat          # 检查服务状态
│   └── *.bat                # 其他辅助脚本
├── 驱动程序/                  # [可选] 驱动程序目录
│   ├── Logitech/            # 罗技驱动
│   ├── Razer/               # 雷蛇驱动
│   └── SteelSeries/         # 赛睿驱动
└── 说明文档/                  # 详细文档
    ├── 快速开始指南.txt      # 快速安装指南
    ├── 编译指南.md          # 程序编译指南
    └── 常见问题解答.txt      # 故障排除
```

## 快速开始

### 第一步：编译程序
1. 如果您还没有编译好的程序，请先编译：
   - 打开"说明文档/编译指南.md"
   - 按照指南在Windows机器上编译程序
   - 编译后将`CyberCafeUsbManager\bin\Release\`目录下的所有文件复制到"主程序"目录

### 第二步：部署到网吧
1. 将整个"网吧USB管理软件_部署包"文件夹复制到网吧机器
2. 建议路径: `C:\网吧软件\USB管理软件\`

### 第三步：安装服务
1. 进入"脚本工具"目录
2. **右键点击"安装服务.bat"**
3. **选择"以管理员身份运行"**
4. 按照提示完成安装

### 第四步：测试功能
1. 插入USB键盘或鼠标（罗技、雷蛇、赛睿等）
2. 查看事件查看器确认识别成功
3. 检查驱动是否自动安装

## 重要提示

### 编译要求
- 程序需要在Windows环境编译
- 需要.NET Framework 4.7.2 SDK
- 可使用Visual Studio或命令行编译

### 运行要求
- Windows 10/11 操作系统
- .NET Framework 4.7.2运行时
- 管理员权限（安装时）

### 网吧环境适配
- 已修复编码乱码问题（所有脚本使用UTF-8）
- 提供简化测试脚本（不依赖编译工具）
- 支持顺网系统集成
