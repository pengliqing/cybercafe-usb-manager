# 网吧USB管理软件 - 编译指南

## 环境要求

### 方案一：Windows 10/11 系统（推荐）
- **操作系统**: Windows 10/11 (64位)
- **.NET Framework**: 4.7.2 或更高版本
- **编译工具** (任选其一):
  - Visual Studio 2019/2022 (社区版免费)
  - Visual Studio Build Tools 2019+
  - .NET SDK 6.0+ (用于 `dotnet build` 命令)

### 方案二：Linux/Mac 系统 (使用Mono)
- **操作系统**: Ubuntu/CentOS/macOS
- **Mono运行时**: 6.12+ (包含msbuild)
- **安装命令**:
  ```bash
  # Ubuntu/Debian
  sudo apt install mono-complete
  
  # CentOS/RHEL
  sudo yum install mono-complete
  
  # macOS
  brew install mono
  ```

## Windows 编译步骤

### 方法一：使用Visual Studio（图形界面）
1. **安装Visual Studio**:
   - 下载 [Visual Studio 2022 社区版](https://visualstudio.microsoft.com/zh-hans/downloads/)
   - 安装时选择"使用C#的桌面开发"工作负载
   - 确保包含.NET Framework 4.7.2 SDK

2. **打开项目**:
   - 双击 `CyberCafeUsbManager.sln` 文件
   - 等待Visual Studio加载项目

3. **编译项目**:
   - 在顶部菜单选择"生成" → "重新生成解决方案"
   - 或按 `Ctrl+Shift+B` 快捷键
   - 选择"Release"配置和"Any CPU"平台

4. **检查输出**:
   - 编译成功后，在 `CyberCafeUsbManager\bin\Release\` 目录中会生成:
     - `CyberCafeUsbManager.exe` (主程序)
     - `CyberCafeUsbManager.exe.config` (配置文件)
     - `Newtonsoft.Json.dll` 等依赖库

### 方法二：使用命令行（PowerShell/CMD）
1. **安装编译工具**:
   ```powershell
   # 方法A: 安装Visual Studio Build Tools
   # 下载地址: https://visualstudio.microsoft.com/zh-hans/downloads/?q=build+tools
   # 安装"使用C++的桌面开发"和".NET Framework 4.7.2 SDK"
   
   # 方法B: 安装.NET SDK
   # 下载地址: https://dotnet.microsoft.com/zh-cn/download
   ```

2. **打开命令行并进入项目目录**:
   ```cmd
   cd /d "C:\Users\你的用户名\Desktop\cybercafe-usb-manager"
   ```

3. **编译项目**:
   ```cmd
   :: 使用MSBuild (Visual Studio自带)
   "C:\Program Files (x86)\Microsoft Visual Studio\2019\BuildTools\MSBuild\Current\Bin\MSBuild.exe" CyberCafeUsbManager.sln /p:Configuration=Release /p:Platform="Any CPU"
   
   :: 或使用dotnet CLI
   dotnet build CyberCafeUsbManager.sln -c Release
   ```

4. **验证编译结果**:
   ```cmd
   dir CyberCafeUsbManager\bin\Release\
   ```

### 方法三：使用批处理脚本
项目已提供自动编译脚本:
```cmd
:: 以管理员身份运行
右键点击 scripts\build_and_test.bat → "以管理员身份运行"
```

## Linux/Mac 编译步骤 (使用Mono)

### 1. 安装Mono
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install mono-complete

# CentOS/RHEL
sudo rpm --import "https://keyserver.ubuntu.com/pks/lookup?op=get&search=0x3FA7E0328081BFF6A14DA29AA6A19B38D3D831EF"
sudo yum install yum-utils
sudo yum-config-manager --add-repo https://download.mono-project.com/repo/centos8-stable.repo

# 验证安装
mono --version
msbuild --version
```

### 2. 编译项目
```bash

cd cybercafe-usb-manager

# 使用msbuild编译
msbuild CyberCafeUsbManager.sln /p:Configuration=Release /p:Platform="Any CPU"

# 或使用xbuild (旧版Mono)
xbuild CyberCafeUsbManager.sln /p:Configuration=Release /p:Platform="Any CPU"
```

### 3. 检查输出
```bash
ls -la CyberCafeUsbManager/bin/Release/
```

## 常见问题解决

### 1. 编译失败: "未找到.NET Framework 4.7.2"
- **解决方案**: 安装.NET Framework 4.7.2开发者包
- **下载地址**: https://dotnet.microsoft.com/download/dotnet-framework/net472

### 2. 编译失败: "缺少NuGet包"
- **解决方案**: 恢复NuGet包
  ```cmd
  nuget restore CyberCafeUsbManager.sln
  ```
  或使用Visual Studio的"还原NuGet包"功能

### 3. 编译成功但运行时报错
- **可能原因**: 缺少依赖库
- **解决方案**: 确保以下文件在同一目录:
  - CyberCafeUsbManager.exe
  - Newtonsoft.Json.dll
  - log4net.dll
  - TaskScheduler.dll
  - CyberCafeUsbManager.exe.config

### 4. 网吧环境测试问题
- **编码乱码**: 已修复，所有批处理脚本添加了 `chcp 65001`
- **权限不足**: 以管理员身份运行程序
- **缺少.NET运行时**: 安装.NET Framework 4.7.2运行时

## 预编译部署包准备

### 部署包结构
```

├── 主程序/
│   ├── CyberCafeUsbManager.exe          # 编译好的主程序

│   ├── 卸载服务.bat
│   ├── 控制台测试.bat

├── 驱动程序/                           # 常用USB驱动（可选）
│   ├── Logitech/
│   ├── Razer/
│   └── SteelSeries/
└── 说明文档/
    ├── 快速开始指南.txt
    ├── 编译指南.md (本文档)
    └── 常见问题解答.txt
```

### 创建部署包
1. 将编译好的文件复制到 `主程序` 目录
2. 复制配置文件到 `配置文件` 目录
3. 复制脚本工具到 `脚本工具` 目录
4. 添加驱动程序（如果需要）
5. 添加说明文档
