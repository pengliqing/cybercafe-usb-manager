# 网吧USB管理软件 - 网吧测试指南

## 📦 文件清单
本压缩包包含：
- `CyberCafeUsbManager/` - C#源代码（需编译）
- `scripts/` - 测试脚本
- `usb_device_database.json` - USB设备数据库
- `README.md` - 项目说明
- `TEST_PLAN.md` - 详细测试计划

## 🚀 快速开始（网吧环境）

### 方案A：已有编译工具（推荐）
如果网吧测试机已安装Visual Studio或.NET SDK：

1. **解压文件**到桌面或D盘
2. **以管理员身份**打开PowerShell
3. **运行编译测试**：
   ```powershell
   cd 解压后的路径\cybercafe-usb-manager
   scripts\build_and_test.bat
   ```
4. 按照脚本提示完成测试

### 方案B：无编译工具
如果网吧机器没有开发环境：

1. **安装最小编译工具**（需要网络）：
   ```powershell
   # 在PowerShell中运行（管理员）
   # 安装Chocolatey包管理器
   Set-ExecutionPolicy Bypass -Scope Process -Force
   [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072
   iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
   
   # 安装.NET SDK
   choco install dotnet-sdk -y
   ```
   
2. **或手动安装**：
   - 下载 [.NET SDK 6.0](https://dotnet.microsoft.com/download/dotnet/6.0)
   - 运行安装程序
   - 重启PowerShell

3. **然后执行方案A的步骤**

## 🔧 网吧环境特殊注意事项

### 1. 管理员权限
网吧可能限制管理员权限。如果无法获取：
- 联系网管协助
- 使用网吧管理卡登录
- 或选择"用户模式"测试（功能受限）

### 2. 还原系统
网吧通常有还原系统，重启后文件会丢失：
- **测试前**：确保文件保存在非还原分区（如D盘）
- **测试后**：及时保存测试结果和日志
- **部署时**：需要考虑持久化方案

### 3. 顺网系统集成
如需测试顺网集成：
1. 确认顺网版本号
2. 测试服务能否随顺网启动
3. 检查与顺网的兼容性

### 4. 网络限制
网吧可能限制下载和网络访问：
- 驱动在线下载功能可能受限
- 准备本地驱动包备用
- 测试离线模式功能

## 📋 测试重点（网吧环境）

### 必须测试的项目：
- [ ] **USB设备识别**：插入罗技、雷蛇等设备，看能否识别
- [ ] **驱动安装**：测试自动安装驱动功能
- [ ] **软件启动**：测试品牌配套软件自动启动
- [ ] **服务稳定性**：长时间运行（2-4小时）
- [ ] **多设备支持**：同时插入多个设备

### 顺网专项测试：
- [ ] **启动项添加**：服务能否随系统/顺网启动
- [ ] **还原系统兼容**：重启后服务是否正常
- [ ] **性能影响**：CPU/内存占用是否可接受

## 🐛 问题排查

### 常见问题1：编译失败
```
错误：未找到MSBuild或dotnet
```
**解决**：
- 安装.NET SDK（见方案B）
- 或使用网吧已有Visual Studio

### 常见问题2：权限不足
```
拒绝访问。请以管理员身份运行。
```
**解决**：
- 右键点击 → "以管理员身份运行"
- 联系网管获取权限

### 常见问题3：服务安装失败
```
服务创建失败（5：拒绝访问）
```
**解决**：
- 确认杀毒软件没有拦截
- 检查Windows Defender设置
- 临时禁用安全软件测试

### 常见问题4：设备无法识别
```
未检测到USB设备
```
**解决**：
- 确认USB接口正常
- 检查设备驱动是否已安装
- 测试不同的USB端口
